How To Play

    Game Objective:

    The goal of the game is to guess the secret code within a certain time limit. The code consists of a sequence of digits.

    Game Elements:

    • Level: The current level is displayed at the top of the game area. Each level increases the difficulty by adding more digits to the secret code.
    • Enter your guess: Enter your guess in the input field provided. Your guess should be a numerical sequence with the same number of digits as the secret code.
    • Check Button: Click the "Check" button to check your guess.
    • Result: The result of your guess will be displayed below. It shows how many digits are in the correct position and have the correct value, as well as how many digits have the correct value but are in the wrong position.
    • Timer: A timer displays the remaining time for each level. If you run out of time, the game is over.

    Gameplay:

    1. Start Game: When you first open the page, the game starts automatically. The level is set to 1, and you have a certain amount of time to guess the secret code.
    2. Guessing: Enter your guess in the input field and click "Check" to submit your guess.
    3. Feedback: After each guess, the game will provide feedback on how many digits are correct in terms of position and value.
    4. Level Up: If you guess the secret code correctly within the time limit, you'll advance to the next level. The code length and time limit will increase for the next level.
    5. Game Over: If you run out of time, the "Game Over!" popup will appear, allowing you to restart the game.

    Restarting the Game:

    If you want to restart the game at any point, click the "Restart" button on the "Game Over!" popup.

    Winning the Game:

    To win the game, you must successfully guess the secret code for all levels.

    Note:

    - Ensure your guesses are valid by entering a numerical sequence with the same number of digits as the secret code.
    - The timer is your enemy; try to guess the code before time runs out.
    - Enjoy the challenge of progressively longer secret codes as you advance through the levels.
